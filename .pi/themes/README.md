# LUMEN Theme

Theme name: `lumen-bi-midnight`

Concept: **LUMEN — Midnight Intelligence**
A dark, calm BI/Evidence dashboard theme for data-driven projects.

## Palette

- Deep navy base (`#0B1020`)
- Luminous cyan identity (`#7DD3FC` / `#38BDF8`)
- Restrained violet intelligence accents (`#A78BFA` / `#8B5CF6`)
- Amber warnings (`#FBBF24`)
- Mint success (`#34D399`)
- Coral error (`#F87171`)

## Activate

```json
{
  "theme": "lumen-bi-midnight"
}
```

## Hot reload

Edit `.pi/themes/lumen-bi-midnight.json` while Pi is running — it reloads automatically.
