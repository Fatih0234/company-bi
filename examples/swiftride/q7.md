# Weekend vs. Weekday Patterns — Should SwiftRide tailor strategy by day?

> Question 7 of 7 from the analysis intention for **swiftride-nyc-playbook**.

**Bottom line:** **Yes — weekend and weekday demand are different markets.** Weekdays peak at 5–6 PM (commuter rush), weekends peak 3–6 PM with stronger late night. SwiftRide should run **different zone/hour strategies** for each.

---

## Weekend vs Weekday Volume

How does overall demand compare?

```sql q7_overall_comparison
SELECT 
  CASE 
    WHEN EXTRACT(DOW FROM c.tpep_pickup_datetime) IN (0, 6) THEN 'Weekend'
    ELSE 'Weekday'
  END as day_type,
  COUNT(*) as trip_count,
  ROUND(AVG(c.total_amount), 2) as avg_total,
  ROUND(SUM(c.total_amount), 0) as total_revenue,
  ROUND(AVG(c.trip_distance), 2) as avg_distance,
  ROUND(AVG(c.tip_amount), 2) as avg_tip
FROM (
  SELECT * FROM files.yellow_tripdata_2024_01
  WHERE fare_amount > 0 AND trip_distance > 0 AND trip_distance < 100
    AND passenger_count > 0
) c
GROUP BY 1
```

<DataTable data={q7_overall_comparison}/>

**Pattern:** Weekdays have 2.5M trips ($68M revenue) vs 690K weekend trips ($18M). But weekend avg fare is slightly lower ($26 vs $28) and trips are shorter (3.3mi vs 3.4mi).

## Hourly Pattern by Day Type

When does demand peak on weekdays vs weekends?

```sql q7_hourly_by_day
SELECT 
  CASE 
    WHEN EXTRACT(DOW FROM c.tpep_pickup_datetime) IN (0, 6) THEN 'Weekend'
    ELSE 'Weekday'
  END as day_type,
  EXTRACT(HOUR FROM c.tpep_pickup_datetime) as pickup_hour,
  COUNT(*) as trip_count,
  ROUND(AVG(c.total_amount), 2) as avg_total
FROM (
  SELECT * FROM files.yellow_tripdata_2024_01
  WHERE fare_amount > 0 AND trip_distance > 0 AND trip_distance < 100
    AND passenger_count > 0
) c
GROUP BY 1, 2
ORDER BY 1, 2
```

<LineChart data={q7_hourly_by_day} x=pickup_hour y=trip_count series=day_type title="Weekday Peak at 5-6 PM; Weekend Peak at 3-6 PM with Later Night" />

**Key differences:**
- **Weekday:** Sharp peak at 5–6 PM (150K trips), morning rush 8–10 AM
- **Weekend:** Broader peak 3–6 PM (45K trips), elevated late night (10 PM–2 AM)
- **Weekend late night** has 33K trips at midnight vs 35K weekday — nearly equal despite 3.6x fewer total weekend trips

## Zone Patterns by Day Type

Do different zones dominate on weekends vs weekdays?

```sql q7_zone_by_day
SELECT 
  z.Zone,
  CASE 
    WHEN EXTRACT(DOW FROM c.tpep_pickup_datetime) IN (0, 6) THEN 'Weekend'
    ELSE 'Weekday'
  END as day_type,
  COUNT(*) as trip_count,
  ROUND(AVG(c.total_amount), 2) as avg_total
FROM (
  SELECT * FROM files.yellow_tripdata_2024_01
  WHERE fare_amount > 0 AND trip_distance > 0 AND trip_distance < 100
    AND passenger_count > 0
) c
JOIN files.taxi_zone_lookup z ON c.PULocationID = z.LocationID
WHERE z.Zone IN ('Midtown Center', 'Times Sq/Theatre District', 'East Harlem North', 'JFK Airport', 'Upper East Side South')
GROUP BY 1, 2
ORDER BY 1, 2
```

<BarChart data={q7_zone_by_day} x=Zone y=trip_count series=day_type title="Times Square and JFK Stay Busy on Weekends" />

**Zone behavior:**
- **Midtown Center:** 115K weekday → 25K weekend (4.6x drop)
- **Times Sq/Theatre District:** 80K weekday → 23K weekend (3.5x drop) — weekend holds better
- **JFK Airport:** 112K weekday → 25K weekend (4.5x drop) — airport traffic weekday-heavy
- **East Harlem North:** 10K weekday → 2K weekend — green cab territory, weekday dominant

## Day-of-Week Breakdown

Which specific days are strongest?

```sql q7_day_of_week
SELECT 
  CASE EXTRACT(DOW FROM c.tpep_pickup_datetime)
    WHEN 0 THEN 'Sunday'
    WHEN 1 THEN 'Monday'
    WHEN 2 THEN 'Tuesday'
    WHEN 3 THEN 'Wednesday'
    WHEN 4 THEN 'Thursday'
    WHEN 5 THEN 'Friday'
    WHEN 6 THEN 'Saturday'
  END as day_name,
  EXTRACT(DOW FROM c.tpep_pickup_datetime) as dow,
  COUNT(*) as trip_count,
  ROUND(AVG(c.total_amount), 2) as avg_total,
  ROUND(SUM(c.total_amount), 0) as total_revenue
FROM (
  SELECT * FROM files.yellow_tripdata_2024_01
  WHERE fare_amount > 0 AND trip_distance > 0 AND trip_distance < 100
    AND passenger_count > 0
) c
GROUP BY 1, 2
ORDER BY 2
```

<BarChart data={q7_day_of_week} x=day_name y=trip_count title="Wednesday Is Peak; Sunday Is Slowest" />

**Day rankings:** Wednesday (459K) > Tuesday (426K) > Thursday (397K) > Saturday (384K) > Friday (379K) > Monday (372K) > Sunday (306K)

## Weekend Night Hotspots

Where do weekend late-night trips concentrate?

```sql q7_weekend_night_zones
SELECT 
  z.Zone,
  z.Borough,
  COUNT(*) as late_night_trips,
  ROUND(AVG(c.total_amount), 2) as avg_total
FROM (
  SELECT * FROM files.yellow_tripdata_2024_01
  WHERE fare_amount > 0 AND trip_distance > 0 AND trip_distance < 100
    AND passenger_count > 0
    AND EXTRACT(DOW FROM tpep_pickup_datetime) IN (0, 6)
    AND EXTRACT(HOUR FROM tpep_pickup_datetime) >= 22
) c
JOIN files.taxi_zone_lookup z ON c.PULocationID = z.LocationID
GROUP BY z.Zone, z.Borough
ORDER BY late_night_trips DESC
LIMIT 10
```

<BarChart data={q7_weekend_night_zones} x=Zone y=late_night_trips series=Borough title="Weekend Late Night: Midtown and Times Square Dominate" />

**Weekend night zones:** Midtown Center, Times Sq, Upper East Side — same Manhattan core, but different trip purpose (entertainment vs commuting).

## Findings

- **Weekdays = commuter market:** Sharp 5–6 PM peak, morning rush, higher volume
- **Weekends = entertainment market:** Broader afternoon peak, strong late night, lower volume
- **Weekend late night is a distinct market:** 10 PM–2 AM has equal trips to weekdays despite 3.6x fewer total weekend trips
- **Zone behavior differs:** Midtown drops 4.6x on weekends; Times Sq holds better (3.5x)
- **Day rankings:** Wednesday peak, Sunday slowest
- **Strategy implication:** Run weekday commuter shifts (7–10 AM, 4–8 PM) and weekend entertainment shifts (2–10 PM)

---

## Notes

- Weekend = Saturday + Sunday (DOW 0 and 6)
- Late night = 10 PM–2 AM
- Weekend late night trips likely include bar/restaurant traffic
- Zone patterns may shift seasonally (summer weekends vs winter)
