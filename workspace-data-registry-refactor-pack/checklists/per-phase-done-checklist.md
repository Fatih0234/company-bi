# Per-Phase Done Checklist

After each implementation phase:

- [ ] Files changed are limited to the phase scope.
- [ ] No unrelated refactors were made.
- [ ] Existing behavior still works.
- [ ] New behavior is covered by tests or manual smoke checks.
- [ ] Generated files are placed in the correct location.
- [ ] No secrets were read or printed.
- [ ] No raw data is published by default.
- [ ] The agent reports files changed and tests run.
